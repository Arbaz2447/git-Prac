Platform Name: SmartLearn




1. Platform Overview
SmartLearn is an online EdTech platform that provides structured, self-paced professional courses. The platform focuses on clear learning workflows, transparent assessment rules, progress visibility, and automated certification.
SmartLearn is designed to help learners understand how learning progresses, how assessments work, and how certificates are earned, without providing assessment answers or academic tutoring.


2. Available Courses on SmartLearn
SmartLearn currently offers the following courses:
Full Stack Web Development
Data Structures & Algorithms
Python for Beginners
Introduction to Cloud Computing
Each course follows a standardized structure for consistency and clarity.


3. Course Structure (Common Across All Courses)
Each course is divided into Modules
Each module contains multiple Lessons
Lessons include:
Video lectures
Reading materials
Practice quizzes
Modules must be completed in order
Quizzes are mandatory to unlock the next module

5. Learning Workflow
Learner enrolls in a course
First module becomes available
Lessons within the module are completed
Module quiz is attempted
On passing the quiz, the next module unlocks
Final assessment becomes available after all modules are completed
6. Assessments (Process Only)
Quiz Rules
Each module has one mandatory quiz
Minimum passing score: 60%
Maximum attempts per quiz: 3
Quiz attempts are tracked automatically


Final Assessment
Required for course completion
Available only after all modules are completed
Passing is mandatory for certification
Assessment Limitations
Answers and solutions are not provided in documentation
No hints, grading rubrics, or sample responses are shared
7. Progress Tracking
Learners can view progress from the SmartLearn Dashboard.
Dashboard Displays
Enrolled courses
Completed lessons
Completed modules
Quiz status (passed / not passed)
Overall course completion percentage
Completion Logic
Lessons must be viewed to count as completed
Modules are completed only after quiz pass
Course completion requires all modules and final assessment
8. Certification Workflow
Certificate Eligibility
Certificates are issued only after:
All modules are completed
All module quizzes are passed
Final assessment is cleared
Certificate Generation
Certificates are auto-generated
Generated within 24 hours after course completion
Linked to the learner’s account
Certificate Access
Available in the Certificates / Achievements section
Can be downloaded after issuance
